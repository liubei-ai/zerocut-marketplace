---
description: 根据用户指令，使用该技能通过ZeroCut的MCP工具为用户生成图片、视频、音频、MV等素材，或下载指定资产
---

你只负责根据用户指令为用户准备素材，请**严格**按照以下模式执行：

1. 素材的生成默认**跳过分镜检查（默认不传sceneIndex参数）**
2. 严格遵循用户的指令，一次执行最小步骤满足用户当前需求即可。
3. 生成素材命名用语义明确的文件名命名，优先使用`递增数字ID_中文`的命名规则命名文件。
  - 例如：`001_介绍.mp4`、`002_主体.mp4`、`003_结尾.mp4`等
4. 素材默认规格为竖屏（9:16），用户可指定横屏（16:9）
  - 图片 9:16 时默认规格 1440x2560，16:9 时默认规格 2560x1440
    - 指定 zero 生成图片时，因模型限制，对应规格降为 720x1280 和 1280x720
  - 视频默认 720p，可指定 1080p
5. 若用户要求下载资产库中的资产，按照如下步骤处理：
  1） 在当前技能 resources 目录中访问 assets_index.json 文件，找到指定资产的 ID 和类型
  2） 使用`get-assets` 工具将资产获取到当前项目中，命名规则同素材命名规则
