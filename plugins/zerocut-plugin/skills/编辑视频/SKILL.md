# 约束

- 除非用户特别说明，模型一律使用 `seedance-2.0`
- 除非用户特别说明，生成视频的规格（aspect_ratio、resolution）参数**不指定**，默认为与原视频相同

# 编辑视频

## 修改视频中的内容

当用户说将视频中某内容替换为另外的内容时，将该视频以及参考图（如有）作为 `generate-video` 方法的参考素材（videos和images）传入，模型会根据用户指令生成新的视频。

### 用户指令例子
  - 将视频1礼盒中的香水替换成图像1中的面霜，运镜不变

## 拼接多个视频

当用户将两段或多个视频拼接时，将这些视频作为 `generate-video` 方法的参考素材（videos）传入，模型会根据用户指令生成新的视频。

### 用户指令例子
  - 视频1中的拱形窗户打开，进入美术馆室内，接视频2，之后镜头进入画内，接视频3
